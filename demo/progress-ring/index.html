Progress Ring

Progress Ring is a Solar2D plugin that lets you easily create customisable vector-based progress rings (or circles). Simply set the progress value and the ring updates automatically.


The source code for this project has not yet been made public. It might be on its way, but you can always contact Eetu and ask for it directly.
