Weaver

Spyric Story Weaver is a narrative and dialogue engine for Solar2D with Twine 2 integration. It consists of a custom Twine 2 story format that exports interactive narratives as JSON, and a Lua runtime that parses and executes the exported stories in Solar2D games.

Features include variables, conditional blocks, navigation links, style markup, and dynamic code execution within story passages.


The source code for this project has not yet been made public. It might be on its way, but you can always contact Eetu and ask for it directly.
