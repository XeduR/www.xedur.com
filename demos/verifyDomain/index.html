←   verifyDomain

verifyDomain is a simple module that will verify if the HTML5 project is running on an approved domain. This is a simple DRM measure that will prevent people from simply downloading your Solar2D games from sites like itch.io and copying them to their own sites.

This demo project isn't all that exciting as it checks if it is running on an approved domain, which it is, so it will always succeed.
