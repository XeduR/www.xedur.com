←   The Dark

The Dark is my COMPO entry for Ludum Dare 46. The theme is "Keep it alive".

This entire game and all of its assets were created within 48 hours. Well, closer to 15-16 hours as I kept sleeping and doing other stuff, like taking naps, a bit too much.

HOW TO PLAY:
- Use A/D or arrow keys to move left and right.
- Use S to toggle SFX on/off.
- Use SPACE to interact with objects.
- Use Q to restart.

This game requires a keyboard to be played, so it will most likely not work on your mobile device.
