Autotile

autotile is a simple Lua module that determines what tile should be placed to a selected location. Upon placing a tile, the module will also update all surrounding tiles as necessary. The GUI is developed using Solar2D.

  • Source available at: https://github.com/XeduR/Solar2D-Projects/tree/master/autotile
